# 🎣 Phishing Attack Simulation & Network Analysis

## 📌 Overview

This project simulates phishing attacks in a controlled lab environment to understand attacker techniques and analyze network traffic generated during the attack.

## 🎯 Objectives

* Simulate phishing attacks using Social-Engineer Toolkit (SET)
* Capture and analyze network traffic
* Identify indicators of compromise (IoCs)

## 🛠️ Tools & Technologies

* Social-Engineer Toolkit (SET)
* Wireshark
* Splunk
* Kali Linux

## ⚙️ Methodology

1. Created a phishing attack scenario using SET
2. Captured network traffic using Wireshark
3. Imported logs into Splunk for analysis
4. Identified suspicious HTTP requests and credential capture patterns

## 📊 Results

* Successfully simulated phishing attack in lab environment
* Detected credential harvesting attempts
* Analyzed traffic patterns and attack behavior

## 🔐 Key Learnings

* Understanding phishing attack flow
* Network traffic analysis using Wireshark
* Log monitoring and visualization using Splunk

## ⚠️ Disclaimer

This project is for educational purposes only and was conducted in a controlled environment.
